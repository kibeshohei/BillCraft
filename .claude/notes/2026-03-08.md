## 気づき・メモ

- Node.js が v18 では Next.js 16 が動かない。nvm で v20 に切り替え後 node_modules を入れ直しが必要
- DNS を Google（8.8.8.8）に変更しないと MongoDB Atlas の SRV レコードが解決できなかった
- Turbopack 環境では mongoose の `models` を分割代入するとundefinedになる → `mongoose.models?.["User"]` で回避
- Zod v4 でエラー取得が `.errors` → `.issues` に変更されている
- Client Component 内で `"use server"` は使えない → actions.ts に切り出す
- `useSearchParams()` は `<Suspense>` でラップ必須
